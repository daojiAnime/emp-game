# 这是一个横板闯关游戏

## 介绍

这是使用Python的Pygame框架编写一个小游戏，有地图系统，人物操作，二段跳，重力模拟，敌人系统，积分系统。基于Youtube博主开源游戏改编，非原创素材。

> main.py --> 基础完整版程序
>
> main_net.py -->  局域联网版
>
> map_editor.py --> 地图编辑器
>
> ... 其余为资源文件和开发过程拆分代码。

## 使用

`pip install pygame`

`python main.py`

关于python基础运行程序操作。
